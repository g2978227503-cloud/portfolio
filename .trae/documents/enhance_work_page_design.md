# Plan: Enhance Work Page Design (Goofy Style)

目标：在保持内容清晰的基础上，通过 Goofy 设计语言（系统美学、交互反馈、视觉张力）提升 `work.html` 的设计感。

## 1. 核心交互：悬停图像预览 (Hover Image Reveal)
目前列表纯文字过于单调。我们将实现“鼠标悬停项目时，浮现对应项目视觉图”的效果，增加沉浸感。

- **HTML/CSS**:
  - 在页面底层（或固定区域）添加 `.preview-container`。
  - 为每个 `.work-row` 添加 `data-preview="url..."` 属性（使用 Unsplash 占位图）。
  - 样式：预览图默认隐藏，Hover 时通过透明度渐变显示。位置可以是“跟随鼠标”或“固定在屏幕右侧/背景”。建议**固定在右侧 40% 区域**或**作为背景蒙版**，保持 Goofy 的“分栏/画布”感。
- **JS**:
  - 监听 `.work-row` 的 `mouseenter` / `mouseleave`。
  - 切换预览图 URL，并触发显示动画（简单的 CSS opacity transition）。

## 2. 列表排版“系统化” (Systemic List Styling)
增强“档案/数据库”的感觉，而非普通的网页列表。

- **视觉样式**:
  - **Hover 效果**：悬停时，列表项背景变色（如深灰/反色），左侧出现装饰性光标（如 `>` 或 `█`）。
  - **Meta 信息**：将右侧的标签（SYS, AIGC 等）样式化为“胶囊”或“代码块”风格（Mono 字体，边框，小字号）。
  - **分割线**：增强网格感，确保边框清晰锐利。

## 3. 顶部统计区 (Stats) 动效
- **计数动画**：加载时，数字从 0 滚动到最终值（05, 10+, 30+）。
- **装饰**：给 Stats 容器增加“角标”或“扫描线”背景。

## 4. 氛围装饰 (Atmosphere)
- **ASCII 装饰**：在 Group Header 旁增加随机的 ASCII 字符装饰。
- **页脚**：保持与 About 页一致的系统风格（已完成）。

## 执行步骤

1.  **结构改造**: 为 `work.html` 的列表项添加图片数据属性，并插入预览容器 DOM。
2.  **样式升级**: 编写 CSS，实现新的列表 Hover 态、预览容器样式、Meta 标签样式。
3.  **脚本增强**: 在 `work.html` 底部（或更新 `app.js`）添加预览交互逻辑和数字滚动逻辑。
4.  **验证**: 检查交互流畅度与移动端适配（移动端通常禁用悬停预览）。
