01 — PROJECT OVERVIEW

品牌 IP 动画与延展

基于 BIGO LIVE 的品牌 IP 形象，打造一系列高质量的动画视频与周边延展设计，提升品牌在全球市场的认知度与影响力。

02 — SOCIAL MEDIA

社媒 IP 动画

结合全球热点与节日节点，创作了一系列生动有趣的 IP 短视频，在 Facebook、Instagram 等社交媒体平台广泛传播，有效提升了品牌的活跃度与用户粘性。

03 — MERCHANDISE

04 — GLOBAL TRAVEL

结合不同国家的地标与文化特色，打造 BIGO DINO 环球旅行系列视觉，展现品牌的全球化视野与多元文化包容性。
