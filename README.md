# Goofy Portfolio

Personal portfolio website built with Goofy Design System.

## Features
- **Goofy Design System**: Minimalist, technical, and editorial aesthetic.
- **Responsive**: Adapts to desktop and mobile screens.
- **Interactive**: Hover effects, text scrambling, and smooth transitions.

## Tech Stack
- HTML5
- CSS3 (Variables, Grid, Flexbox)
- Vanilla JavaScript

## Local Development

1. Clone the repository.
2. Open `index.html` in your browser or run a local server:
   ```bash
   python3 -m http.server 8000
   ```
