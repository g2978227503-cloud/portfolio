大神APP品牌升级

全站视觉语言重塑与品牌资产沉淀
CLIENT
网易大神
ROLE
视觉负责人 / 品牌设计
YEAR
2025
OUTPUT
品牌识别 / UI规范 / 运营物料
01 — BACKGROUND

重塑游戏社区的视觉语言

随着大神APP用户群体的扩张和功能的迭代，原有的视觉体系已无法满足多元化的业务需求。品牌形象老化、视觉一致性缺失，导致用户认知模糊。

本次品牌升级旨在通过构建全新的视觉语言系统，强化“游戏热爱者社区”的品牌定位，提升用户的归属感和产品的易用性。

02 — CORE CONCEPT

核心理念：多元共生与价值感知

基于“多元共生”的品牌核心，重新定义了视觉关键词。通过对社区、漫剧、赛事及会员四大业务板块的深度拆解，提炼出差异化的视觉特征。

从社区的“活人感”到漫剧的“沉浸感”，再到赛事的“专业感”与会员的“荣耀感”，每一板块都拥有独特的视觉识别基因，共同构建起大神 APP 丰富而统一的品牌生态。

03 — DESIGN SYSTEM

评分榜视觉体系重构

针对大神 APP 核心的“评分榜”业务，建立了一套高度模块化且具备情感连接的视觉体系。通过“活人感”的核心概念，将真实的玩家声音与评分数据相结合。

从基础的品牌基因提取到多维度的评分卡片设计，再到针对《梦幻西游》《蛋仔派对》《燕云十六声》《永劫无间》等不同游戏 IP 的定制化榜单适配，实现了视觉风格的统一性与业务场景的灵活性。

04 — MANJU THEATER

大神漫剧：沉浸式内容剧场

从“内容流”向“专属剧场”的体验升级。通过数字光影艺术与焦点化设计，打造漫剧专属的“沉浸式剧场”氛围。

结合品牌基因塑造“上映感”焦点图，强化漫剧心智与瞬时流量的吸引力，让用户在浏览过程中获得身临其境的观看体验。

05 — ESPORTS LIVE

赛事直播：专业竞技体验

通过将品牌符号进行“竞技化”转译，结合暗黑质感与动态光效，打造大神核心赛事的专业化感知。

针对 NBPL、IVL、高校联赛等不同赛事 IP，定制专属的直播间、数据榜单及选手评分界面，强化电竞氛围，让用户沉浸于巅峰竞技的激情与荣耀之中。

06 — SUPER VIP

超级会员：荣耀身份进阶

基于 Liquid 设计语言，重构 VIP 品牌符号与视觉识别体系。通过金属质感与流光动效的结合，塑造尊贵而独特的身份印记。

从 V5 到 V7，构建了差异化的权益感知体系。专属的会员徽章、礼包界面及特权中心，让用户在每一次交互中都能感受到“身份进化”带来的荣耀感与归属感。
