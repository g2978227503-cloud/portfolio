01 — PROJECT OVERVIEW

产品概述

Deepkeep 是一款整合了饮食管理、运动课程与训练日志核心功能的 AI 健康管理平台。旨在通过 AI 动态生成个性化健身计划，实现“训练-饮食-数据追踪”的全链路闭环。

目标用户： 健身用户、有健身意向但无计划人群以及需要健康管理的老年化人群。

02 — CORE FUNCTIONS

一期核心功能

1. AI 动态训练计划生成

  • 数据输入： 支持用户基础信息（BMI、体脂率、运动能力）及目标（增肌/减脂/塑形）录入，并支持 Apple Watch 体脂率、心率等可穿戴设备数据自动同步。
  • 动态生成： 基于 DeepSeek 通义模型 + 关键词训练，结合用户画像生成包含有氧类型、器械选择、组数/次数的个性化计划。（如：减脂用户生成“HIIT+力量训练”组合）。
  • 动态调整：
    ○根据用户完成状态确认内容与难度。
    ○若未完成当日训练，系统自动缩短后续休息日或降低强度。
    ○根据体脂率变化动态更新食谱（如体脂下降 5% 后增加碳水比例）。

2. 日历集成管理

  • 一键导入： 用户确认计划后，自动生成 iCal 文件并同步至 Apple Calendar，包含训练时间与动作详情。
  • 动态更新： 计划调整后，通过 iOS Shortcuts API 自动更新日历事件并推送通知。
